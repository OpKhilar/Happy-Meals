# Happy Meals

A simple static restaurant landing page built with HTML and CSS.

## Overview

This project showcases a promotional landing screen for a food service called **Happy Meals**. The page uses a full-height background image, a branded content card, and a call-to-action button.

## Files

- `index.html` - Main page markup.
- `style.css` - Custom styling for layout, typography, and the landing card.

## Features

- Responsive landing section with a full-screen background image
- Branded header and subtitle
- Styled call-to-action button
- Google Fonts integration for modern typography
- Bootstrap CDN included for layout helpers and optional utility classes

## Usage

1. Open `index.html` in your browser.
2. The page does not require a local server or backend.

## Notes

- The background image is loaded from an external URL.
- Text on the content card uses a dark font color (`#323f4b`) for contrast against the yellow card background.
- Bootstrap scripts and styles are included from CDN for potential extension.
- The current design is a static prototype and can be expanded into a full restaurant website.
